Package managers simplify dependency management in software projects. They automate the process of installing, updating, and maintaining libraries and tools required for your project.

## Installation

To install dependencies using **Bun**, first, ensure Bun is installed (typically via `curl https://bun.sh/install | bash` or `npm install -g bun`).

Once installed, navigate to your project root and execute:

```bash
bun install
```

This will install all the dependencies defined in the `package.json` file efficiently, leveraging Bun's speed.
